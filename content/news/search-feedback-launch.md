# 中文资料检索与反馈入口说明

当内容逐渐变多之后，用户最容易遇到的问题就不再是“有没有资料”，而是“我该去哪里找”。因此，搜索、FAQ、反馈和社区入口应该被一起理解。

## 更适合的查找顺序

1. 先用站内搜索按关键词找资料
2. 再看 FAQ 是否已有类似问题
3. 仍然找不到时提交反馈
4. 需要公开讨论的问题再进入 GitHub

## 为什么这样更省时间

- 能先解决大部分重复问题
- 不会一上来就把所有事情都扔进 GitHub
- 方便中文用户逐步进入更公开的协作渠道
