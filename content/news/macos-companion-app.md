OpenClaw 今日发布 macOS 伴侣应用 Beta 版，专为 macOS 15+ 用户设计。

## 主要特性

- **菜单栏快速访问** - 随时从屏幕顶部快速调用助手
- **系统深度集成** - 与 macOS 系统完美融合
- **Apple Silicon 优化** - 充分利用 M 系列芯片性能
- **原生体验** - 遵循 Apple 人机交互指南

## 系统要求

- macOS 15.0 或更高版本
- Apple Silicon 或 Intel 处理器
- 已安装 OpenClaw Gateway

## 安装方式

```bash
# 通过 Homebrew 安装
brew install openclaw

# 或从官网下载
openclaw app install
```

## 使用方法

1. 启动伴侣应用
2. 登录 OpenClaw 账户
3. 即可从菜单栏快速访问助手
