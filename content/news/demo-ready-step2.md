# FAQ 与订阅入口整理

很多用户不是缺一个功能，而是缺一条长期跟踪信息的路径。FAQ 和订阅入口的价值，就在于降低“下次我去哪里看更新”的成本。

## 更适合长期使用的方式

- 常见问题先集中到 FAQ
- 版本和专题更新通过 RSS 或订阅入口跟踪
- 需要补充信息时再进入反馈与 GitHub

## 对中文用户有什么帮助

- 不需要反复手动检查有没有新内容
- 第一次提问前就能先看到高频问题
- 有利于把零散问题逐步沉淀成更稳定的资料
