## 技术生态

OpenClaw 采用现代化的技术栈，主要使用语言包括 TypeScript、Nix、Python 和 Elixir。项目支持 50+ 集成，包括：

- **通讯平台**：WhatsApp、Telegram、Discord、Slack、Signal、iMessage
- **AI 模型**：Claude、GPT 及本地模型
- **生产力工具**：Spotify、Hue、Obsidian、Twitter、Browser、Gmail、GitHub

## 安装方式

OpenClaw 提供多种安装方式：

```bash
# 一键安装
curl -fsSL https://openclaw.ai/install.sh | bash

# npm 全局安装
npm i -g openclaw

# pnpm 安装
pnpm install
```
