## 重要更新

### 产品功能

#### Canvas 交互界面

全新的 Canvas 功能让用户可以：
- 可视化构建工作流
- 拖拽式 Agent 配置
- 实时预览和调试

#### 语音唤醒

iOS 端支持语音唤醒功能：
- "Hey Claw" 唤醒词
- 后台持续监听
- 低功耗优化

#### 多模型支持

扩展的模型支持：
- GPT-4 Turbo
- Claude 3 系列
- 本地模型 (Ollama)

### 渠道集成

新增渠道：
- 飞书 (Feishu)
- Signal
- VirusTotal 集成
- Tailscale 原生支持

### 开发者体验

- CLI 工具增强
- API 文档完善
- SDK 更新 (Python, TypeScript)
