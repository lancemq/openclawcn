OpenClaw 于 2026年3月9日发布重要版本更新，带来多项功能改进和安全增强。

## 主要更新内容

### 性能优化

- Gateway 响应速度提升 30%
- 消息处理流程优化，减少延迟
- 内存占用进一步降低

### 全新技能系统

- 技能市场（ClawHub）已收录 **349** 个开源技能
- 技能版本管理更便捷
- 技能安全验证集成 VirusTotal

### 安全性改进

- 与 VirusTotal 合作推出技能安全验证
- 传输加密增强
- 本地认证支持生物识别

## 安装更新

```bash
# 一键更新
openclaw update

# 或重新安装
curl -fsSL https://openclaw.ai/install.sh | bash
```
