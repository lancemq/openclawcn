# 如何开始系统理解 OpenClaw

第一次接触 OpenClaw 时，很容易因为文档类目较多而失去方向。更稳妥的方式，是先建立一个"从定位到配置"的阅读顺序。

## 推荐顺序

1. 先看产品定位和适用场景
2. 再看核心能力总览
3. 再进入安装、onboarding 和 Control UI
4. 最后再看 channels、hooks、安全和更新跟踪

## 为什么这样更适合第一次阅读

- 能先搞清楚整体地图
- 不会一开始就被配置项淹没
- 读到后面时更容易知道哪些内容和自己有关
