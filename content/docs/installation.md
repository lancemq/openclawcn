# 安装与环境

这一页不负责覆盖所有细节，而是先给中文用户一个清晰的准备顺序。

## 你需要先确认的内容

- 当前使用的操作系统
- Node.js 版本与包管理器
- 是否需要访问 OpenClaw 官方仓库与相关依赖源
- 本地是否具备运行示例和调试环境

## 推荐阅读顺序

1. 先阅读快速入门，理解 OpenClaw 是什么

3. 最后进入更具体的使用教程和最佳实践
