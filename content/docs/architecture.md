# 内容结构说明

这个站点的目标不是展示建设过程，而是把 OpenClaw 相关信息按更适合中文用户理解的方式组织起来。

## 内容优先

- 首页优先回答“OpenClaw 是什么”
- 文档优先回答“如何理解和开始使用”
- 新闻优先承接版本更新、使用提醒和解读

## 中文用户优先

- 术语尽量用中文解释
- 安装与使用路径按中文用户习惯组织
- 社区、反馈和 GitHub 入口保持清晰可见

## 技术边界清晰

- 首页、新闻和文档优先由内容驱动
- 搜索、反馈和订阅负责辅助用户找到内容
- 部署平台属于内部实现，不作为对外内容主题
