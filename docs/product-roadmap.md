# 产品与开发规划

## 当前状态

站点已经具备：

- 面向中文用户的 OpenClaw 文档、新闻、最佳实践和视频教程
- 站内搜索、反馈、RSS、sitemap、robots 和 Vercel 部署能力
- 基于 Nuxt Content 的内容化站点结构

下一阶段的重点不再是单页补充，而是把站点继续做成长期运营型内容产品。

## 目标

1. 让用户更快进入正确学习路径
2. 让文档、视频、新闻、实践形成统一主题消费体验
3. 让反馈真正成为内容缺口和产品优化的输入
4. 继续提升 SEO、内容组织和发布质量

## 已落地方向

- 学习路径页：`/paths`
- 主题中心页：`/topics`
- 首页分流与本周推荐
- 视频页平台 / 难度 / 主题筛选
- 反馈表单分类增强
- Node 22 + Nuxt Content native sqlite 固定

## 后续可继续推进

- 主题详情页独立化，而不只通过查询参数聚合
- 视频页增加内嵌预览或缩略图
- 反馈结果自动聚合成待补内容清单
- 热门内容与最近更新自动打分
- 每个主题的专属 OG 图与结构化数据
