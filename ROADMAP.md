# OpenClawCN 开发路线图

> 最后更新: 2026-03-10

## 项目现状

**OpenClawCN** 是基于 Nuxt 3 + Vercel 的 OpenClaw 中文官网。

### 已完成模块

| 模块 | 状态 | 数量 |
|------|------|------|
| 首页 | ✅ 完成 | 6个核心区块 |
| 文档系统 | ✅ 完成 | 5个分类目录 |
| 新闻系统 | ✅ 完成 | 24篇文章 |
| 最佳实践 | ✅ 完成 | 36篇文章 |
| 基础 SEO | ✅ 完成 | robots/sitemap/rss |
| API 接口 | ✅ 完成 | 5个端点 |
| 内容管理 | ✅ 完成 | 校验+清单生成 |

---

## 第一阶段：功能完善（✅ 已完成）

**目标**: 提升核心功能可用性

**时间**: 2026-03-10 完成

| 优先级 | 任务 | 状态 | 说明 |
|--------|------|------|------|
| P0 | 全文搜索实现 | ✅ 完成 | 集成 Nuxt Content 本地搜索，支持搜索建议、热门关键词、快捷键 ⌘K |
| P0 | 404页面优化 | ✅ 完成 | 增加推荐链接、搜索入口、热门搜索词 |
| P1 | 文档导航增强 | ✅ 完成 | 添加目录树、上一篇/下一篇导航、快速入口 |
| P1 | 代码高亮优化 | ✅ 完成 | 添加复制按钮、深色主题、语法高亮颜色 |
| P1 | 图片懒加载 | ✅ 完成 | 配置 @nuxt/image，添加 OptimizedImage 组件 |
